# Go Server

A simple Go server without any external libs.

## Running

Make sure you have go installed on your machine, then:

```sh
  go run main.go
```

That's it! :)

## End-Points

Base URL: **localhost:8000**

- **/** - API entry point.

- **/hello** - Hello with optional query param. Eg: **/hello** or **/hello?user=foo**
